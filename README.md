### Instalación y configuración

- #### Local
    * `npm ci`
    * Tener levantada PostgreSLQ en `localhost:5432` con una base de datos "wallets_db"
    * `npm run startdev`

- #### Local con Docker Compose
```
docker-compose up --build
```
