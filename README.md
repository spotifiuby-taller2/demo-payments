### Instalación y configuración

- #### Local
    * `npm ci`
    * Como pre-requisito se debe tener levantada una base de PostgreSLQ que escuche en `localhost:5432`
    * `npm run startdev`

- #### Local con Docker Compose
```
docker-compose up --build
```
